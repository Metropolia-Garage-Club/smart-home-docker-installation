# HACS configuration

Download and install HACS:

https://hacs.xyz/docs/use/download/download/#to-download-hacs

Since we are using docker container select **container** option. After downloading, you should see HACS in the Home Assistant side bar. Go to HACS, search for what you need, and download it.

**Important:** Make sure **advanced mode** is enabled in your Home Assistant profile.

After downloading, you may need to reload (If it doesn't reload automatically)

~~~
Settings -> Dashboard -> Three dots -> /hacsfiles/check-button-card/check-button-card.js............
~~~ 

if you don't see this URL, click **Add resource** and add the newly added card
