# Aqara Presence Sensor

- First, download the **Aqara mobile app**.
- Create an account.
- Select and connect to your wifi (use 2.4GHz band in the aqara app).
- Click **Add Accessory**.
- Select the **Presence sensor**.
- Once it connects to your mobile app, You will see the sensor in Home Assistant.
